# Release Notes

## Version 1.2.0

- Added new command line options to control the viewport size and turn on mobile emulation
- Changed some of the command line options to be more consistent
- Added documentation for calling the test function directly (not just through the CLI or Jest/Cypress)

## Version 1.1.3

- Removed async/await from the [Jest](./jest) and [Cypress](./cypress) assertions
- Additional rule updates

## Version 1.1.2

- Minor bug fix in a [Cypress](./cypress) assertion failure message

## Version 1.1.1

- Added ability to test with [Cypress](./get-started#cypress)
- Additional rule updates

## Version 1.0.3

- First major release
- Option for test results on stdout
- Configurable timeout setting for dom rendering when using CLI

## Version 0.1.21

- Addition of WCAG A/AA/AAA criteria levels in output
